KEEP YOUR VALUABLES IN THE VAULT

The Secure Open Source Password Manager

Vaultron is a cryptographically secure cross platform password manager written in electron and node.js which allows users to store their account information for websites locally with easy access. It will store hashed passwords in a JSON file for safe keeping. Vaultron will allow multiple profiles, each one will have a master password of its own that will unlock The Vault. This master password will let the user not only let them access their passwords, but also edit any information that is stored in The Vault. The master password will be created by the user so that they can remember it. Our password manager also allows the user to create and store strong and secure passwords that are created with our password generator. This allows the user to only need to remember the master password.
